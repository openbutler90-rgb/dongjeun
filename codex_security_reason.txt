Codex Security가 필요한 이유:

동전커피는 Firebase 기반 커뮤니티 플랫폼으로, 관리자 페이지에서 사용자 가입 승인/거절, 공지사항 발송, 푸시 알림 전송 등 민감한 작업을 수행합니다. 또한 Firestore 보안 규칙, 서버 측 Express API, OneSignal 푸시 연동 등 보안이 중요한 코드베이스를 포함하고 있습니다. Codex Security를 활성화하면 AI가 생성하는 코드가 sandbox 환경에서 검증되어, 프로덕션에 영향을 줄 수 있는 위험한 코드 실행(예: 데이터베이스 삭제, 권한 변경, API 키 노출 등)을 사전에 방지할 수 있습니다. 특히 다중 사용자가 접속하는 커뮤니티 앱 특성상, 코드 생성 과정에서의 보안 계층은 필수적입니다.
